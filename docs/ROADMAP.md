- local prompt file parser
- dry run
- side panel controls
- content script messaging

- optional image download
- optional upscale flow
- output folder management
- CSV/JSON export
- Etsy listing metadata generation

## V4: Multi-provider abstraction

- Leonardo.ai adapter
- Ideogram adapter
- local provider abstraction
- OpenAI image provider, if appropriate
